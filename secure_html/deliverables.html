Team Blue Spring 2022

Technical Risk Matrix

Technical Risks

  • T-1: Matching API mismatches user’s art.
  • T-2: Market does not comply with DMCA.
  • T-3: ID API does not correctly verify identity.
  • T-4: System sends DMCA to wrong art.

Technical Risk Mitigations

  • T-1: User verifies art is theirs before sending DMCA.
  • T-2: Flag art that reappears in the market place and notify user.
  • T-3: Verify Information and make corrections.
  • T-4: Final verification check.

Customer Risk Matrix

Customer Risks

  • C-1: Users abuse the system by uploading art they do not own.
  • C-2: Customer doesn’t know how to use the app.

Customer Risk Mitigations

  • C-1: Sign up using art accounts.
  • C-2: Integrated User Tutorial.

Customer Risk Matrix

Security Risks

  • S-1: Unauthorized access to a user account.
  • S-2: Database Integrity.

Security Risk Mitigations

  • S-1: Two Factor Authentication for Login.
  • S-2: Encrypt all user personal data.
